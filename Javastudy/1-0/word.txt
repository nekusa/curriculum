IT
	information technology
	インターネットなどの通信とコンピュータとを屈指する情報技術

ICT
	information and communication technology
	情報通信技術
	インターネットなどを経由して人と人とをつなぐ役割

IoT
	Internet of Things
	従来インターネットに接続されていなかった様々なモノが
	ネットワークを通じてサーバーやクラウドサービスに接続され、
	相互に情報交換をする仕組み

OS
	Operating System
	基本システムを全体で管理し、様々なアプリケーションソフトを
	動かすための最も基本的なソフトウェア

Windows
	Microsoft社が提供しているOS


macOS (Mac OS X)
	AppleがていきょうしているOS

UNIX
	現存する中で最も古いOS
	UNIXから派生したOSも多い

Linux
	Unix系OSカーネルであるLinuxカーネルを指し、
	広義には、それをカーネルとして周辺を整備したシステム全体

マークアップ
	文書構造（テキスト）や視覚表現（装飾）などの情報を、
	コンピューターが正しく認識できるように、タイトルや見出しなどの
	各構成要素に「タグ」と呼ばれる識別のための目印を使い、
	意味付けを行うこと

HTML
	HeyperTextMarkupLanguageの略
	主にWebsiteのコンテンツ構造を作るために使うコード

CSS
	CascadingStyleSheets
	HTML や XMLで記述された文書の体裁や見栄えを
	表現するために用いられます。

JavaScript
	オブジェクト指向のスクリプト言語
	Webやバックエンドデスクトップアプリ・
	モバイルアプリ・サーバーなど利用される場面は多岐にわたる。

jQuery
	JavaScriptの書き方を簡単にすることができるJavaScriptで作成されたライブラリ
	Web系では、ブラウザによる違いを気にしなくてよい

JavaSE
	Java Platform Standard Edition
	Javaで使用されるAPIをまとめたものです。

.NET
	一般的に.NET Frameworkと呼ばれるアプリケーションを指す
	WindowsのシステムやWebサービスなどの開発に利用されている

Java
	オブジェクト指向言語
	プラットフォーム非依存であるのが特徴

JavaEE
	Java Platform, Enterprise Edition
	サーバマシン、ワークステーション向けのJava

Perl
	C言語やsed、awk、シェルスクリプトなど他の
	プログラミング言語の優れた機能を取り入れている
	
PHP
	サーバーサイドで動的なウェブページを作成するための機能を多く備えている

Ruby
	オブジェクト指向スクリプト言語
	国産で有名なプログラミング言語

Apache
	全世界的に普及している無償で利用できるオープンソースのWebサーバのソフト

Nginx
	他のWebサーバーのソフトと比べて高速・大量処理が得意

IIS
	Microsoft社が提供するWindowsのWebサーバーソフト

SQL
	RDBMSのソフトウェアを制御する国際基準のデータベース言語のこと

DB2
	IBM社が提供しているRDBMS

MySQL
	世界で最も人気のあるオープンソースデータベース

Maria DB
	MySQLの派生したRDBMS

Oracle DB
	Oracle社のRDBMS

PostgreSQL
	原子性、整合性、独立性、耐久性 (ACID)プロパティを持つトランザクション

SQLite
	サーバーとしてではなくアプリケーションに組み込んで利用されるデータベース

SQL Server
	MicrosoftのAzureのRDBMS

CRUD
	RDB上で、どのデータに対してどのプロセスが
	生成(Create)、読み込み(Read)、更新(Update) 、削除(Delete)
	されるかを表す表形式のマトリックス

トランザクション
	一連の処理を一単位で意味する

コミット
	トランザクション処理がすべて成功し結果を確定すること

ロールバック
	トランザクション処理中にエラーが発生した場合にトランザクション処理開始前の状態に戻すこと

排他制御
	複数のプロセスが利用できる共通資源に対し競合しないように、他プロセスから利用できないようにすること

楽観ロック
	資源を外部から利用できないようアクセス禁止にはしない

悲観ロック
	処理開始時に他の主体によるアクセスを禁じる

実行計画
	SQLserverがユーザーが発行したSQLを解析し決定した実データに対して、どのようにアクセス・計算を行うかが記載されている計画書

チューニング
	性能の悪いSQLを発見し、その処理過程を改善すること

パフォーマンス
	いわゆる性能
	SQLクエリの応答時間など

スロークエリ
	クエリに対する応答時間が一定の基準に照らして遅いこと

クエリログ
	実行されたすべてのSQLを記憶しているログ

IDE（統合開発環境）
	エディタやコンパイラをまとめて提供する開発環境そのもの
